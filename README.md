# @charliewilco/extensions

Web components library built with strict TypeScript, Bun, and Rolldown.

## Install

```bash
bun add @charliewilco/extensions
```

## Usage

```ts
import "@charliewilco/extensions/slider";
```

```html
<x-slider min="0" max="10" value="4"></x-slider>
```
